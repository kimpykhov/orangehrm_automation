# OrangeHRM Test Automation Framework

Production-grade test automation framework for a role-based HR management system (OrangeHRM).

## Goals
- End-to-end UI and API test automation
- Scalable and maintainable test architecture
- Stable execution with flaky test handling
- CI/CD integration

## Tech Stack
- Python
- Playwright
- Pytest
- GitHub Actions

## Scope
- Authentication and role-based access
- Employee management workflows
- Core business scenarios (positive and negative)

## Status
Project initialized. Framework setup in progress.
